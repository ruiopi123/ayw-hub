Getting Started

Aywsion の拡張は、ここから始まります。

01 クイックスタート

1. ターミナルを開く

Aywsion OS、またはデバッグ環境のターミナルを起動してください。

2. ライブラリをインストールする

terminal
ayw install [library-name]

※ [library-name] には Explorer で見つけた名前を入れます。

02 守るべきルール

🔤 命名は英語のみ

インストールの成功確率を 100% に保つため、ライブラリ名に日本語（全角文字）は使用できません。小文字、数字、ハイフンのみを推奨します。

📦 A-Z ソート

すべてのパッケージはアルファベット順に管理されます。これにより、膨大なライブラリの中から直感的に目的の機能を探し出すことが可能です。

03 開発者向けガイド

How to publish?

自作のライブラリが完成したら、Publish ページから登録してください。URLは、GitHubのReleaseリンクなど、直接ファイルをダウンロードできる場所を指定してください。

Success rate

日本語の排除に加え、依存関係を最小限に抑えることで、すべての環境で一貫した動作を保証することを目指しています。
